# AI 旅行规划师需求文档

## 一、核心功能

1.  **智能行程规划**
    *   用户输入：支持语音（强制要求）和文字输入旅行目的地、日期、预算、同行人数、旅行偏好。
    *   AI 输出：自动生成个性化的旅行路线，包含交通、住宿、景点、餐厅等详细信息。

2.  **费用预算与管理**
    *   AI 预算分析：由 AI 进行预算分析。
    *   开销记录：记录旅行开销。

3.  **用户管理与数据存储**
    *   注册登录系统：用户可保存和管理多份旅行计划。
    *   云端行程同步：旅行计划、偏好设置、费用记录等数据云端同步，支持多设备查看和修改。

## 二、提交要求

1.  提交一个 PDF 文件，包含 GitHub 仓库地址和 README 文档。
2.  项目代码提交至 GitHub。
3.  提供可以直接下载运行的 Docker 镜像文件。
4.  提供如何运行的 README 文档。
5.  API Key：如果使用的不是阿里云的 API Key，需将 Key 提交在 README 文档中，并保证 3 个月内可用。
6.  GitHub 提交记录：保留尽可能多且详细的 GitHub 提交记录。

## 三、注意事项

1.  API Key 安全：切勿将任何 API Key 写在代码中，尤其是 GitHub 上公开的代码库。
